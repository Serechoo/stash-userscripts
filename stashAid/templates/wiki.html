Plugin Manager

Overview: The Plugin Manager is a centralized system for managing and updating plugins that extend the functionality and features of your platform. It simplifies the process of installing new plugins, updating existing ones, and managing the overall plugin ecosystem.

How to Use

  • Viewing Available Plugins: Users can view a list of available plugins, including their names, descriptions, versions, and release dates. This helps in deciding which plugins to install.
  • Installing Plugins: To install a plugin, click the 'Install' button next to the plugin listing. The system will automatically handle the download and installation process.
  • Updating Plugins: The 'Update All' button allows users to update all installed plugins to their latest versions with a single click, ensuring that all plugins are current with the latest features and security patches.
  • Reloading Plugins: After installing or updating plugins, the 'Reload Plugins' button can be used to ensure they are properly integrated into the platform without requiring a complete system restart.

Bulk File Renamer

Overview: The Bulk File Renamer tool automates the renaming of files in bulk, based on predefined patterns. It significantly improves file organization and accessibility.

How to Use

  • Selecting Files: Users can select multiple files for renaming based on specific criteria such as file type, date, or other attributes.
  • Defining Rename Patterns: Users can define patterns for renaming files, which may include keywords, dates, or sequences. These patterns ensure consistent naming conventions.
  • Previewing Changes: Before finalizing the rename operation, users can preview the new file names to ensure accuracy and prevent errors.
  • Renaming Files: With satisfaction in the preview, users can proceed to rename the files. The system will apply the new names according to the defined pattern.

Task Scheduler

Overview: The Task Scheduler allows for the scheduling and management of automated tasks. Users can define task details, select scripts for execution, and set times for these tasks to run automatically.

How to Use

  • Defining Tasks: Users can create tasks by specifying a name, description, and the script to execute. This allows for easy identification and management of tasks.
  • Scheduling Tasks: Each task can have a scheduled time for execution, supporting a variety of scheduling options like one-time, recurring, or event-triggered tasks.
  • Managing Tasks: Users can view a list of their scheduled tasks, including details like status and next execution time. Tasks can be edited or deleted as needed.

Stash BIG Data

Overview: The BIG Data feature offers advanced capabilities for viewing and managing large datasets, including filtering, sorting, and pagination.

How to Use

  • Viewing Data: Users can view datasets in tabulated forms, with options for sorting and filtering based on various criteria to quickly find needed information.
  • Filtering and Sorting: Comprehensive filters and sorting options allow users to narrow down datasets and organize the displayed information according to specific needs.
  • Pagination: For large datasets, pagination helps in navigating through data in manageable chunks, enhancing the user experience by preventing overload.

Stash CSS Editor

Overview: The Stash CSS Editor is an integrated tool for editing and customizing the platform's CSS in real-time, featuring syntax highlighting and immediate preview changes.

How to Use

  • Editing CSS: Users can write or modify CSS code within the editor. Syntax highlighting aids in identifying code elements easily.
  • Previewing Changes: Changes made in the CSS editor are immediately reflected on the platform, allowing for rapid customization and iteration.
  • Saving Changes: Once satisfied with the modifications, users can save their changes. The platform will automatically apply the new styles.
